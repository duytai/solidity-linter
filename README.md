# SSA
A collection of utilities to statically analyze solidity program

## 1. Installation
Add to `Cargo.toml`
```
ssa = "0.1.2"
```
### 2. Usage
We have an example in https://github.com/duytai/ssa/blob/master/samples/src/main.rs

### 3. API
https://docs.rs/ssa/0.1.2/ssa/
